Configuration

The configuration variables are the they dials and knobs that adjsut everything. Wowza!

Variable Default What it do
$rem-size 16px Defines the basic size from which all others are derived.
$ratio 1.618 Establishes a ratio for sizing along the typographic scale.
$body-size 1rem Defines the default size of body type.
$small-size 0.85rem Defines a secondary, smaller size for type. Is used to create a double stranded typographic scale.
$baseline 0.25rem Establishes the baseline grid.
$leading 5*$baseline Establishes the default leading for type.
$indent 1em Establishes the standard indent to use when applicable.
$container-ratio 11/12 Defines the proportion of grid width to screen width at small screen sizes.
$container-ratio-medium 10/12 Defines the proportion of grid width to screen width at medium screen sizes.
$container-ratio-large 8/12 Defines the proportion of grid width to screen width at large screen sizes.
$small 30rem When to switch out of small sizing.
$large 60rem When to switch out of medium sizing.
$max-width 60rem The maximum width of the content grid.
$gutter 4% The width of gutters between columns.
$col-count 12 The number of columns to build the grid with.
$padding-x $gutter The default padding of columns on the x-axis.
$padding-y $baseline The default padding of columns on the y-axis.
$padding-outer-x $gutter / 2 The default padding of outside columns on the x-axis.
$padding-outer-y $baseline The default padding of outside columns on the y-axis.
