Type

The type system is the real meat of the situation. Using a modular double stranded type scale with all sorts of good shit.

Relevant Configuration:

Variable Default What it do
$rem-size 16px Defines the basic size from which all others are derived.
$ratio 1.618 Establishes a ratio for sizing along the typographic scale.
$body-size 1rem Defines the default size of body type.
$small-size 0.85rem Defines a secondary, smaller size for type. Is used to create a double stranded typographic scale.
$baseline 0.25rem Establishes the baseline grid.
$leading 5*$baseline Establishes the default leading for type.
$indent 1em Establishes the standard indent to use when applicable.

Typographic System

There are a couple of moving parts here:

  • Font Styles
  • Font Weights
  • Leading
  • Tracking
  • Word Spacing
  • Alignment
  • Type Scale

Font Styles

Here we borrow from classical print typography terminalogy, rather than the not quite accurate web aproach.


// Mixins
@mixin font-roman()
@mixin font-italic()

// Classes
.txt-roman
.txt-italic

Font Weights


// Mixin
@include font-weight(100 || 200 || 300 || 400 || 500 || 600 || 700 || 800)

// Classes
.txt-thin
.txt-light
.txt-regular
.txt-medium
.txt-demi
.txt-bold
.txt-black

// Responsive Classes
.ns-txt-${weight}
.md-txt-${weight}
.lg-txt-${weight}

Leading

Again, refernce the pieces of metal that sit between lines of type in a letterpress layout. This defines the light height of a block of text. Integers are available from 1-12.


// Mixin
@include leading($i)

// Classes
.txt-lead-${i}

// Responsive Classses
.ns-txt-lead-${i}
.md-txt-lead-${i}
.lg-txt-lead-${i}

Tracking

Tracking is the space between letters. Some typographid desicisions benefit from a tight, negative tracking while others (like big, bold sans) can look amazing widely tracked out. The mixin takes an integer, and assigns letterspacing equal to `i*0.001em`


// Mixin
@include tracking($i)

// Classes
.txt-tracked-wide  // @include tracking(20)
.txt-tracked-tight // @include tracking(-10)

// Responsive Classes
.ns-txt-tracked-${wide || tight}
.md-txt-tracked-${wide || tight}
.lg-txt-tracked-${wide || tight}

Word Spacing

The space between the words. Used sparingly, but can be essentiall at times. The mixin takes an integer, and assigns word spacing equal to `i*0.001em`


// Mixin
@include word-spacing($i)

// Classes
.txt-spaced-wide  // @include word-spacing(20)
.txt-spaced-tight // @include word-spacing(-10)

// Responsive Classes
.ns-txt-spaced-${wide || tight}
.md-txt-spaced-${wide || tight}
.lg-txt-spaced-${wide || tight}

Alignment

Left, right, center. Simple stuff! An addtion option of `inline` assign the display property to inline, useful for textual elements that have a default display of `block`, like headers


// Mixins
@include text-center()
@include text-left()
@include text-right()
@include text-inline()

// Classes
.txt-center
.txt-left
.txt-right
.txt-inline

// Responsive Classes
.ns-txt-${center || left || right || inline}
.md-txt-${center || left || right || inline}
.lg-txt-${center || left || right || inline}

Type Scale

Creates 14 or 15 steps along a double stranded typographic scale based on the body font size, the small font size, and the ratio. Classes availble from -3 to 12.


// Mixin
@include font-size($i)

// Class
.txt-size-${i}

// Responsive Classes
.ns-txt-size-${i}
.md-txt-size-${i}
.lg-txt-size-${i}
