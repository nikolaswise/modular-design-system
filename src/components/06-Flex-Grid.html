Flex Grid

The power of grids systems the joy of flexbox. Peep the example below to see it in action.

Relevant Configuration:

Variable Default What it do
$baseline 0.25rem Establishes the baseline grid.
$container-ratio 11/12 Defines the proportion of grid width to screen width at small screen sizes.
$container-ratio-medium 10/12 Defines the proportion of grid width to screen width at medium screen sizes.
$container-ratio-large 8/12 Defines the proportion of grid width to screen width at large screen sizes.
$small 30rem When to switch out of small sizing.
$large 60rem When to switch out of medium sizing.
$max-width 60rem The maximum width of the content grid.
$gutter 4% The width of gutters between columns.
$col-count 12 The number of columns to build the grid with.
$padding-x $gutter The default padding of columns on the x-axis.
$padding-y $baseline The default padding of columns on the y-axis.
$padding-outer-x $gutter / 2 The default padding of outside columns on the x-axis.
$padding-outer-y $baseline The default padding of outside columns on the y-axis.

Container

Contains the grid to the ratios defined by `contiainer-ratio-x` and `max-width` configutation variables.


// Mixin
@inlude container()

// Class
.container

Row

Defines discreet vertical sections.


// Base Mixin
@include row();

// Base Class
.row

// Modifier Mixins
@include row-is-centered();
@include row-is-right();
@include row-is-vertical();
@include row-no-wrap();
@include row-is-v-centered();

// Modifier Classes
.row-is-centered
.row-is-right
.row-is-vertical
.row-no-wrap
.row-is-v-centered

Columns

Creat horizontal divisions within rows. The integers available for row counts are equal to the number of rows defined by the `col-count` configuration variable. The spaces between rowsis defined by the `gutter` confgiuration variable.


// Base Mixin
@include column();

// Base Class
.col

// Modifier Mixins
@include column-is($i)

// Modifier Classes
.col-is-${i}
.ns-col-is-${i}
.md-col-is-${i}
.lg-col-is-${i}

// Adjustment Mixins
@include is-offset($i);
@include float(left || right);
@include flush();

// Adjustment Classes
.is-offset-{$i}
.col-is-left
.col-is-right
.col-is-flush

Leads

The idea of leads (/'l…õds'/, like the type of metal) is taken from the typgraphic layouts practiced in hot-metal typesetting. The lead is a strip of metal that is used to create white space between elements. Here, te leads are used to add margin above block elements in multples of the `baseline` variable to maintain consistentn vertical rhythym.

The integers 1-12 are available for classes.


// Mixins
@mixin lead-top();
@mixin lead-bottom();

// Classes
.lead-${i}
.lead-bottom-${i}
